## Learning Manager

A Productivity App to also serve as a Rails and AngularJS playground.  
Use it to keep track of the Books, Courses and Videos.

![Learning Manager](http://i.imgur.com/SmNiwwE.png)

Makes use of the following tech:

* Ruby on Rails (v3)
* Angular JS
* Bootstrap
* Bower

#### Usage

`rails server`
